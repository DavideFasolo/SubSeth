# SubSeth

Plugin di utility per VCV Rack, da abbinare a submarine per lo stile.


## Attribuzione componenti grafici

Le porte `SilverPort`, `RedPort` e `BluePort` di SubSeth derivano dal codice grafico di SubmarineFree di David O'Rourke, con adattamenti minimi e senza supporto allo stile flat.
